# Programs and Processes

$ yum list * // all or ? s t f
$ yum list available
$ yum search tree

$ yum install tree

$ init // all programs come from init
$ ps 
$ ps -u
$ top (h help k kill q quits r changepriority s updaterate P displaybycpuusage M sortbymemoryusage)
$ cd var/log
$ tail messages
